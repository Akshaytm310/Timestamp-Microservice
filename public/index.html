Timestamp Microservice

Endpoints:

  • /api → current time
  • /api/:date → date string (e.g. 2015-12-25) or Unix ms (e.g. 1451001600000)

Examples

  • /api
  • /api/2015-12-25
  • /api/1451001600000

Expected JSON

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

If input is invalid, response is:

{ "error": "Invalid Date" }

Note: Time zone conversion is not part of this project; dates are parsed with new Date() as GMT.
